# Buttons

## Filled

```
<div>
  <Button>Default</Button>
  <Button intent="primary">Primary</Button>
  <Button intent="secondary">Secondary</Button>
  <Button disabled>Disabled</Button>
</div>
```

## Outlined

```
<div>
  <Button variant="outlined">Default</Button>
  <Button intent="primary" variant="outlined">Primary</Button>
  <Button intent="secondary" variant="outlined">Secondary</Button>
  <Button disabled variant="outlined">Disabled</Button>
</div>
```

## Text Buttons

```
<div>
  <Button variant="text">Default</Button>
  <Button intent="primary" variant="text">Primary</Button>
  <Button intent="secondary" variant="text">Secondary</Button>
  <Button disabled variant="text">Disabled</Button>
</div>
```

## Full Width

```
<Button fullWidth>Full Width</Button>
```
